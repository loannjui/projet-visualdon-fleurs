# Nuancier de fleurs par région

## Contexte
Ce projet a pour but de cartographier la diversité de la flore suisse par région. À l'aide d'une interface exploratoire, il présente les espèces locales et les organise sous forme de nuancier pour identifier les couleurs dominantes de chaque zone géographique.

## Description

## But
L’objectif est d’encourager l’exploration de la nature tout en analysant la répartition des espèces et de leurs caractéristiques chromatiques en fonction de l'altitude et des régions.
